Instructions

  • Phrases with are the command you use to DM the bot
  • Once complete, you can start doing missions.
  • You can follow the guides in the 'EZ MODE' tab on the spreadsheet or on the Timeline webpage to get all of the loot and get ready for the next sector.
  • If you die we all get penalised.

  • Click on the Midnight Protocol bot announcement:
    Get Claws role!
  • You should now have access.
  • You are now able to verify yourself as a Pre-Variant. (Go to m:\verify)
  • Give the bot some time to make the appropriate changes.

  • On the Discord go to:
    #midnight-protocol-spam
  • Type in the following command: M:\Verify @YourDiscordUsername
    (make sure you actually tag yourself in the message)
  • You should now receive a DM from the MidnightProtocol bot asking your for a password.
  • Once you get that DM reply to the bot with this password 14421PCGSPD
  • You should now be a Pre-variant and be able to use any of the valid commands in the bots DM's.
